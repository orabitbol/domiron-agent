# Domiron Content Rulebook

## 🔒 Source of Truth

All content MUST be based ONLY on the provided game JSON.
- No guessing
- No adding features
- No generic phrases
- No "fantasy fluff"

If something is not in the JSON → DO NOT USE IT

---

## 🎯 Content Goal

Create Facebook posts that feel like:
- A real competitive war game
- Urgent
- Strategic
- Rewarding skill, not luck

---

## ⚔️ Tone & Style

- Short sentences only
- Aggressive tone
- Command language (תקוף, שדוד, שלוט, הפקד)
- Speak directly to the player (אתה)
- No "nice" or "friendly" tone

❌ Forbidden tone:
- casual
- funny
- childish
- "משחק כיף"

---

## 🔥 Hooks (MOST IMPORTANT)

First line MUST:
- create urgency OR
- create fear OR
- create curiosity

Examples:
- "הזהב שלך חשוף."
- "עוד 30 דקות — והדירוג משתנה."
- "הוא תקף אותך בלילה."

If the first line is weak → the post is invalid

---

## 📊 Use Real Data

Always include:
- numbers (30 דקות, 90 יום, 12%)
- mechanics (טיק, תורות, שלל, מגן)

❌ Never say:
- "הרבה משאבים"
- "קרב חזק"

✅ Always say:
- "12% שלל לכל תור"
- "3 תורות כל 30 דקות"

---

## 🧠 Content Angles

Every post MUST follow ONE clear angle:
- Battle (תקיפה, עוצמה)
- Economy (משאבים, בנק)
- Competition (דירוגים)
- Tribe (שבט, לחשים)
- Spy (ריגול)
- Progression (ערים, שדרוגים)

Do NOT mix too many angles

---

## 🎨 Visual Logic

The text MUST match the visual:
- Battle → soldiers, weapons, attack
- Economy → gold, mines, resources
- Ranking → leaderboard, crown, dominance

---

## 🚫 Forbidden

- No English sentences (except names)
- No emojis spam
- No fake features
- No long paragraphs

---

## ✅ Definition of Success

A good post:
- grabs attention in 2 seconds
- feels like a real game ad
- uses real mechanics
- makes the player want to act NOW
